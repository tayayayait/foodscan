# 기록 인사이트 정책

마지막 업데이트: 2026-05-15

기록 인사이트는 최근 스캔 제품을 요약해 사용자가 반복되는 위험 요인을 확인하도록 돕는다. 이 값은 의료 판단이 아니라 앱 내부 점수와 검수 상태를 요약한 참고 정보다.

## 계산 항목

| 항목         | 기준                                                                                |
| ------------ | ----------------------------------------------------------------------------------- |
| 평균 점수    | 숫자 점수 산출 가능한 제품만 평균                                                   |
| 점수 가능 수 | `computeScore` 결과가 computable인 제품 수                                          |
| 검수 필요 수 | 점수 산출 불가 또는 `needs_review` 상태 제품 수                                     |
| 등급 분포    | 점수 산출 가능한 제품의 A-E 등급 개수                                               |
| 주의 제품    | 회수 이력, 검수 필요, 개인 기준 경고, 고위험 첨가물, 당류·나트륨·포화지방 과다 제품 |
| 최다 요인    | 가장 많이 반복된 주의 요인                                                          |

## 최다 요인 우선순위

동일 개수로 여러 요인이 반복되면 다음 순서로 표시한다.

1. 회수 이력
2. 고위험 첨가물
3. 당류·나트륨·포화지방 과다
4. 검수 필요
5. 개인 기준 경고

개인 기준 경고는 사용자가 설정한 성분 알림과 식이 선호에 따라 달라진다.

## 화면 동작

`/history`는 다음을 표시한다.

- 평균 점수, 주의 제품 수, 검수 필요 수, 최다 요인
- 최다 요인에 대한 후속 행동 문구
- A-E 등급 분포
- 저장 제품, 최근 스캔 제품, 임시 제품을 합산한 안전 감시 패널
- 전체 제품 목록
- 주의 제품만 보는 필터
- 기록 화면에서 상품 상세로 이동하면 저장된 제품 스냅샷만 표시하고 공공 API, 공개 DB, AI 보강, 대체상품 추천 조회를 다시 실행하지 않는다.
- 홈 화면의 최근 스캔 항목도 기록 화면과 동일한 저장 스냅샷 상세로 이동하며 재조회를 실행하지 않는다.

기록이 없으면 인사이트를 표시하지 않고 빈 상태를 표시한다.
