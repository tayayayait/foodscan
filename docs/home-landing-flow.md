# 홈 랜딩 및 앱 진입 흐름

마지막 업데이트: 2026-05-16

## 목적

`/` 경로는 FoodScan 소개용 랜딩 화면으로 사용한다. 사용자가 `무료로 시작하기`, `무료 시작하기`, `지금 무료로 시작하기` CTA를 누르면 실제 식품 스캔 앱 홈인 `/app`으로 이동한다.

## 화면 배치

- 상단에는 FoodScan 로고와 1차 CTA를 고정 헤더로 배치한다.
- 첫 화면은 좌측 핵심 문구와 우측 앱 분석 화면 이미지를 2열로 구성한다.
- 우측 앱 분석 화면 이미지는 `public/home-hero.webp` 애니메이션 자산을 사용하며, 원본 16:9 비율을 유지해 전체 장면이 보이도록 표시한다.
- 모바일에서는 단일 열로 쌓아 첫 CTA가 먼저 보이게 한다.
- 신뢰 배너, 핵심 기능 카드 3개, 하단 CTA 순서로 배치한다.
- 핵심 기능 카드 이미지는 카드 이미지 영역을 꽉 채우는 cover 방식으로 표시하고, hover 시 카드 테두리와 제목 옆 아이콘에 브랜드 틸 강조 애니메이션을 적용한다.

## 앱 홈

`/app`은 기존 식품 스캔 홈 화면을 유지한다.

- 제품명, 제조사, 바코드 검색 입력
- 바코드 스캔 진입 카드
- 최근 스캔 목록 또는 빈 상태 안내

앱 내부 사이드바의 `식품 스캔` 브랜드 텍스트는 `/` 랜딩 홈으로 이동한다. 상단 앱 제목의 `식품 스캔` 텍스트와 하단 내비게이션의 홈 항목은 `/app` 앱 홈을 가리킨다. `/`는 외부 진입용 랜딩 화면으로 분리한다.
