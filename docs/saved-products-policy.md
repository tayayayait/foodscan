# 저장 제품 정책

마지막 업데이트: 2026-05-13

저장 제품은 사용자가 제품 상세에서 북마크한 제품 스냅샷이다. 저장 기능은 서버 계정 기능이 아니라 브라우저 로컬 저장소 기반 개인 보관함이다.

## 저장 기준

| 항목 | 기준 |
| --- | --- |
| 저장 위치 | `localStorage`의 `kfs:saved-products` |
| 저장 단위 | 제품 스냅샷과 `savedAt` 시각 |
| 중복 기준 | `product.id` |
| 정렬 | 최근 저장 순 |
| 최대 보관 수 | 50개 |

같은 제품을 다시 저장하면 기존 항목을 갱신하고 목록 맨 앞으로 이동한다.

## 화면 동작

- 제품 상세의 북마크 버튼은 저장/해제를 토글한다.
- 저장된 제품은 북마크 아이콘을 채워 표시한다.
- `/history`의 `저장` 탭은 저장 제품만 표시한다.
- 저장 제품은 최근 스캔 목록에서 사라져도 제품 상세 조회 대상에 포함된다.

## 데이터 관리

저장 제품은 로컬 데이터 삭제 시 함께 삭제된다. 서버 동기화, 계정 간 공유, 백업은 현재 범위가 아니다.
