# 제품 조회 보강 정책

## 바코드 후보 선택

식품안전나라 `C005`는 하나의 바코드에 여러 품목제조보고번호를 반환할 수 있다.
동일 바코드 후보가 여러 개이면 첫 번째 행을 그대로 사용하지 않는다.

1. 각 `C005` 후보의 `PRDLST_REPORT_NO`로 원재료를 조회한다.
2. `INDUTY_NM`에 `축산물`이 포함된 후보는 `C006` 축산물품목제조보고(원재료)를 우선 조회한다.
3. 그 외 후보는 `C002` 식품(첨가물)품목제조보고(원재료)를 우선 조회한다.
4. 우선 원재료 API가 비면 다른 원재료 API를 fallback으로 조회한다.
5. 원재료 항목 수가 가장 많은 후보를 우선 선택한다.
6. 항목 수가 같으면 원재료 원문 길이가 긴 후보를 선택한다.
7. 그래도 같으면 `C005` 원본 순서를 유지한다.

품목제조보고번호가 확보되면 `CertImgListServiceV3` HACCP 제품이미지 및 포장지표기정보 API를 추가 조회한다.
포장지 원재료가 기존 원재료보다 더 상세하거나, 기존 원재료가 `식품첨가물` 같은 포괄 표기를 포함하고
포장지 원재료는 더 구체적인 경우 포장지 표기를 우선 사용한다. 이 API는 포장지 알레르기 표기와 공식 제품 이미지도
함께 보강한다.

이 기준은 신라면 `8801043014809`처럼 동일 바코드가 여러 제조소 품목보고번호에 연결되고,
일부 품목보고번호의 원재료가 `면류, 과채가공품, 복합조미식품`처럼 요약된 경우를 보정하기 위한 것이다.
빙그레 바나나맛 우유 `88002798`처럼 `축산물가공업-유가공업`으로 분류되는 제품은 `C002`가 비어 있고
`C006`에 원재료가 있는 경우가 있으므로 축산물 원재료 API를 별도 적용한다.

## 첨가물 감지

상세 원재료가 확보되면 각 원재료명을 `I2520` 식품원재료코드로 조회해
`식품첨가물(B코드)` 또는 중분류 `식품첨가물` 여부를 판별한다.
원재료 문자열이 포괄 표기 `식품첨가물`만 포함하면 실제 첨가물명으로 확정하지 않는다.

식품안전나라 인증키는 같은 키의 동시 요청을 거부할 수 있으므로,
첨가물 감지용 원재료 조회는 병렬 호출하지 않고 순차 처리한다.

## 상품 이미지 보강

바코드 상세 조회는 점수 산출 데이터와 이미지 표시 데이터를 분리한다.
이미지는 점수 근거가 아니며, `Product.imageUrl` 표시용 값으로만 사용한다.

이미지 우선순위:

1. 검증 DB 또는 사용자 제출 데이터에 이미 저장된 `Product.imageUrl`
2. Open Food Facts 바코드 단건 조회의 `image_front_url` 또는 `image_url`
3. 네이버 쇼핑 검색 API의 이름 매칭 썸네일

공공 API 제품명과 쇼핑 상품명은 표기가 다르므로 비교 전 괄호 내용, 특수문자, 법인 표기를 제거한다.
예를 들어 `신(辛)라면`과 `(주)농심`은 `신라면`, `농심`으로 비교한다.
네이버 쇼핑 후보는 정규화된 제품명 포함 관계와 브랜드 일치 여부로 선택하며, 매칭되지 않으면 이미지를 붙이지 않는다.

대체상품 추천은 영양 DB 후보 자체에 이미지 컬럼이 없으므로 최종 추천 후보에 한해 제품명 기준 네이버 쇼핑 검색을 추가 수행한다.
카테고리 검색어만으로 얻은 쇼핑 결과는 후보명과 매칭될 때만 사용한다.

## 외국어 상품 정보 한국어 표시

- 공공 API 포장 표기, Open Food Facts, AI OCR 후보처럼 외국어 원재료·카테고리·알레르기 텍스트가 포함된 제품은 최종 후보 선택 후 `translateProductToKorean` Edge Function action으로 한국어 표시용 번역을 시도한다.
- 바코드가 없는 로컬 임시/OCR 제품도 외국어 원재료가 있으면 상세 화면 진입 시 같은 한국어 표시용 번역을 시도한다.
- 번역은 Gemini 구조화 응답을 사용하며 제품명, 카테고리, 원재료 원문, 원재료 배열, 알레르기 배열만 대상으로 한다.
- 영양성분, 바코드, 출처, 신뢰도, 회수 정보, 첨가물 코드는 번역 과정에서 변경하지 않는다.
- Gemini 번역 실패, Edge Function 미설정, API 키 누락 시 조회를 실패시키지 않고 원문을 그대로 표시한다.
- 원재료와 알레르기 배열은 원본 순서와 개수를 유지한다. Gemini 응답에서 일부 항목이 누락되면 해당 항목은 원문을 유지한다.
